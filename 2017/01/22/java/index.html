#关于java语言final修饰符的用法

##为什么需要final修饰符？

final 的意思是最终或者最后的意思。final可以修饰 变量 ， 方法 ，以及 类 。
在我们程序中，有些类或者类中的成员方法或者变量不希望别人来修改。所以，我们通过 Final 来修饰。

>
class A
{
//圆周率不可以被修改；
final float pai=3.1415926f;
}

##final使用注意事项：

  • 给方法修饰，则该方法不可以被重载或者覆盖
  • 给变量修饰时，则该变量不可以被修改
  • 给类修饰时候，则该类不可以被继承
