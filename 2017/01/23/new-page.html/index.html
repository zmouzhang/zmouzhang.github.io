关于排序和查找

一，常用的排序算法

在本文中，列出三种排序方法： 冒泡 、选择 、快速三种排序方法。

1，冒泡排序

冒泡排序就是通过一层层的遍历，然后比较两个数之间的大小，从而选择合适位置进行插入，从而使数组变成有序数组。

public void sort(int arr [])
        {
            int temp=0;
            for(int i=0;i<arr.length-1;i++)
            {
                for(int j=0;j<arr.length-1-i;j++)
                {
                    if(arr[j]>arr[j+1])
                    {
                        temp=arr[j];
                        arr[j]=arr[j+1];
                        arr[j+1]=temp;
                    }
                }
            }
        }

2,选择排序

选择排序首先得找到所有数中的最小数，然后遍历余下的数与最小数比较，然后根据结果来确定每一个数字存放的位置。（可以是从小到大，也可以是从大到小）。

public void ChooseSort(int arr[])
            {
                int temp=0;
                for(int j=0;j<arr.length-1;j++)
                {
                    int min=arr[j];
                    int minIndex=j;
                    for(int k=j+1;k<arr.length;k++)
                    {
                        if(min>arr[k])
                        {
                            min=arr[k];
                            minIndex=k;
                        }
                    }
                    temp=arr[j];
                    arr[j]=arr[minIndex];
                    arr[minIndex]=temp;
                }

}

3,快速排序

快速排序是我们经常使用的排序方法。它是通过空间作为代价，换取快速的一种排序方法。

  • 1．i =L; j = R; 将基准数挖出形成第一个坑a[i]。

  • 2．j–由后向前找比它小的数，找到后挖出此数填前一个坑a[i]中。

  • 3．i++由前向后找比它大的数，找到后也挖出此数填到前一个坑a[j]中。

  • 4．再重复执行2，3二步，直到i==j，将基准数填入a[i]中。

其中，i，j分别是指向数组的开头和尾。

void quick_sort(int s[], int l, int r)  
{  
    if (l < r)  
    {  
        //Swap(s[l], s[(l + r) / 2]); //将中间的这个数和第一个数交换 参见注1  
        int i = l, j = r, x = s[l];  
        while (i < j)  
        {  
            while(i < j && s[j] >= x) // 从右向左找第一个小于x的数  
                j--;    
            if(i < j)   
                s[i++] = s[j];  

          while(i < j && s[i] < x) // 从左向右找第一个大于等于x的数  
              i++;    
          if(i < j)   
              s[j--] = s[i];  
      }  
      s[i] = x;  
      quick_sort(s, l, i - 1); // 递归调用   
      quick_sort(s, i + 1, r);  
  }  
}

二，二分查找

二分查找：选取数组中的中间值，作为第一次比较数。如果要查找的数小于比较数，则从第一个数到比较数的前一位的数所在区间来查找。如果大，则从比较数的后一位数到数组末尾所在区间来查找。依次类推，直到找到位置。

public void BinaryFind(int leftIndex,int rightIndex,int val,int arr [])
    {
        int midIndex=(rightIndex+leftIndex)/2;
        int midVal=arr[midIndex];
        if(rightIndex>=leftIndex)
        {
            if(midVal>val)
            {
                BinaryFind(leftIndex, midIndex-1, val, arr);
            }

        else if(midVal<val)
        {
            BinaryFind(midIndex+1, rightIndex, val, arr);
        }
        else if(midVal==val)
        {
            System.out.println("找到所需要值得下标："+midIndex);
        }
    }
}
